## Usage

### Changing Theme
Click the moon/sun icon in the navigation bar to toggle between light and dark modes.

### Changing Language
Use the language dropdown in the navigation bar to switch between English and French.

### Adding More Languages
Edit `translations.js` and add new language objects with translated strings.
